# DevOps Task Scheduler (Web UI)

Tiny web interface to keep a list of operational tasks/commands.

## Features
- Zero-dependency HTTP server
- Add tasks from a browser
- Stores tasks in JSON
- Good starting point for a richer scheduler

## Usage
```bash
python3 app.py
# open http://localhost:8080
```

## Disclaimer
Current MVP stores task metadata only and does not auto-run commands.
